Dependencies Required:-
1. Python > 3.5 
2. Django install. To install django cmd (pip install django)
3. Install Django-Rest-Framework. To install django-rest-framework cmd (pip install djangorestframework)

How to use:-
1. Open Cmd in directory which contains manage.py

2. Type cmd (python manage.py makemigrations)

3. Type cmd (python manage.py migrate)

4. Type cmd (python manage.py runserver).

5. Open localhost:8000 in your browser.

Note:- 1.command written in ()  paste directly to cmd.
2. if view display This Field is required. Pass that field in url. for ex- localhost:8000/?field=value
